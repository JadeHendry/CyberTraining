# Setup
1. Download and install Rufus [https://rufus.ie/en/#download]
2. Download the latest ISO file for the operating system of choice
3. USB with at least 32 GB storage capacity
---
# Installation
1. Insert USB drive
2. Open Rufus
3. Under `Device` select the USB you want to use
    * If the device does not show up right away, expand the `Show advanced drive properties` dropdown and select `Show USB Devices`
4. Under `Boot Selection` ensure "Disk or ISO image" is selected
5. Beside the `Boot Selection` dropdown, `Select` the ISO file you wish to install
6. In the `Volume Label` field, name your USB device
7. Leaving all other settings default, click `Start`
    * **This will erase everything on your USB device!!!**
---
# Usage
1. Leaving your USB drive inserted, `Restart` your computer
2. During boot, press the corresponding key to view the `boot menu` or `BIOS`
    * *The key and BIOS process will vary based on your computer make/model. Consult your owner's manual for specific instructions*
3. In the boot selection menu, select the USB you just created
